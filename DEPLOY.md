# Deployment Guide — VaultCeaser (Self-Hosted, Windows)

## Architecture

```
Browser → nginx/port 80 (optional) → Next.js :3456 → FastAPI :8080
                                                     ↓
                                               data\db.sqlite
```

Everything runs on the same machine. No external services.

**Disk usage:**
- node_modules + .next build: ~430 MB
- SQLite DB: starts at 8 KB
- Total new: ~430 MB (well under your 3 GB free)

---

## One-Time Setup

Double-click `setup.bat` **or** run in PowerShell:

```powershell
powershell -ExecutionPolicy Bypass -File setup.ps1
```

This will:
1. Create `data\db.sqlite` (the database)
2. Install Python + Node deps
3. Build Next.js
4. Create SQLite tables
5. Write `web\.env` with a random session secret

---

## Start Everything

Double-click `start.bat` **or**:

```powershell
powershell -ExecutionPolicy Bypass -File start.ps1
```

Opens two windows (FastAPI on :8080, Next.js on :3456).  
Then visit `http://localhost:3456`.

---

## Run as Background Services (optional)

Using [NSSM](https://nssm.cc/) (Non-Sucking Service Manager):

```cmd
nssm install vc-api python "C:\path\to\vaultceaser\server.py"
nssm set vc-api AppDirectory C:\path\to\vaultceaser
nssm install vc-web cmd "/c npm start"
nssm set vc-web AppDirectory C:\path\to\vaultceaser\web
nssm set vc-web AppEnvironmentExtra DATABASE_PATH=C:\path\to\vaultceaser\data\db.sqlite BACKEND_URL=http://localhost:8080 SESSION_SECRET=your_secret
nssm start vc-api
nssm start vc-web
```

Or just keep using `start.bat` — it's fine for personal use.

---

## Environment Variables (web\.env)

Auto-created by `setup.ps1`. Edit if needed:

```env
BACKEND_URL=http://localhost:8080
DATABASE_PATH=C:\path\to\vaultceaser\data\db.sqlite
SESSION_SECRET=<auto-generated 64-char hex>
```

---

## nginx (optional, port 80 instead of 3456)

Install [nginx for Windows](https://nginx.org/en/docs/windows.html), copy `nginx.conf` to nginx's `conf\` folder, then:

```cmd
nginx -t
nginx
```

Site available at `http://localhost` (port 80).

```powershell

npm install
npm run build
# restart start.bat
```
